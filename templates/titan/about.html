About

This Project is a proof of concept for INTEX at Brigham Young University.

Features of this Website Include

  • Home Page
    • Ted Ex Video about opioid addiction
    • General Statistics about the opioid crisis
    • Count of deaths per state (Top 5 States)
    • Live Count of Opioids (updates automatically when data is added to the database)
    • Contact Button for comments and further help
  • About
    • General Information about the devlopment team and this website
  • Search
    • Search By Presriber name, state, credential, speicality, & Gender (All Nullable) (Generates Interactable list)
    • Search By Drug Name and if it is an opioid or not (Generates Interactable list)
    • Add a Prescriber (Takes to a new page)
    • Generate a list of answers to common questions
  • Prescriber Detail Page
    • View important inforamtion about a prescriber
      • Name
      • State
      • Specialty
      • Gender
      • Credentials
    • Create prescription information
    • Update prescription information
    • Delete prescriber information
    • View most prescribed drugs
    • View percentage of presciptions that are opioids
    • View total count of prescriptions for opioid and non-opioid drugs
    • Navigate to drug detail pages for prescribed drugs
  • Drug Detail Page
    • View drug information
    • View top 10 prescribers of the given drug
      • Prescriber name
      • Whether they can prescribe opioids or not
      • Total quantity they have prescribed
